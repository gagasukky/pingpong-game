# ⚡ PING PONG GAME

ブラウザで動くクラシックなピンポンゲームです。PC・タブレット・スマホから遊べます。

## 操作方法

| 端末 | 左パドル (Player 1) | 右パドル (Player 2 / AI) |
|------|---------------------|--------------------------|
| **PC** | `W` / `S` キー | `↑` / `↓` キー |
| **タブレット** | 左半分をタッチ＆ドラッグ | 右半分をタッチ＆ドラッグ |

## ゲームルール

- 先に設定した点数（5・11・21点）を取ったプレイヤーの勝利
- ボールはラリーが続くほど加速する
- AI は Easy / Normal / Hard の3段階

## ゲームモード

- 🤖 **1P vs AI** — コンピュータと対戦
- 👥 **2P 対戦** — 同じ端末で2人対戦
